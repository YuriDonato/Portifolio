    Etapa 3
Criando seções do site
    <container> - pega meio do site
    <container-fluid> pega o tamanho total full
	<row> serve para encapsular as col
	<col> colunas podemos determinar seu tamanho col-lg-5

A formula é
    Uma div contendo row e col, nessa ordem
    Dentro da col fica a informação, caso necessario
    Dentro da col outra row, e ai vai

    Col = colunas
    Row = linhas

    Etapa 2
Utilizar SASS para estilizar o html

Usar custom theme colors - adicionando mais cores ao bootstrap

// importa do Bootstrap
@import '../node_modules/bootstrap/scss/_functions';
@import '../node_modules/bootstrap/scss/_variables';

// cria as cores personalizadas
$custom-theme-colors: (
    "verde": #c0f64d,
    "preto": #141620,
);

//adiciona aos arquivos do Bootstrap
$theme-colors: map-merge($custom-theme-colors, $theme_colors);
